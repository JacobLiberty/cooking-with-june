# Cooking with June 🐱🍳

A pastel, homemade-but-painstakingly-crafted digital cookbook for Jacob &
Lily — and June, the orange cat who supervises. Built with Next.js and Sanity,
deployed on Vercel.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm start` — run the production build
- `npm run lint` — lint
- `npm test` — run the test suite (Vitest)
- `npm run test:watch` — Vitest in watch mode

## Environment

Copy `.env.example` to `.env.local` and fill in values as later phases add them
(Sanity, auth). Phase 1 needs no environment variables.
